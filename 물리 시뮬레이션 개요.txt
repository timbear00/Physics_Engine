[물리 시뮬레이션 개요 및 계획]
1. 주제 : 물체의 운동 및 역학적 활동을 표현하기 위한 시뮬레이션 제작
2. 목적 : 운동의 시각화를 통한 물리 현상 이해
3. 방법 :
	- Processing 언어를 사용하여 시각화 진행
	- 객체를 생성하고 물리 법칙을 적용하여 실제 현실세계와 최대한 비슷하게.(delta time 적용)
	- 학습자료답게 보기 쉬운 UI, 각각의 상황마다 속도, 가속도 등의 물리량 표현.
	- 공식으로 인한 결과, 그래프, 실제 시뮬레이션의 상황을 비교
	- 시간이 된다면 이를 웹사이트에 적용하는 것까지. (ps5.js 이용?)
	- Git + Github 이용하여 버전관리 => 웹 호스팅까지
	- 잘 되면 도메인도 하나 살 계획
4. 계획서, 일지 작성하기
